# movie-recommender
# About
A quick python project connecting to OMDb API to get movie recommendation based off inputted prompt on mood. Core concept is the simplification of the process of finding moviees to watch. It aims to eliminate the stress of scrolling through streaming platforms trying to find the movie that your mood at that moment is gravitating towards.

# Deployed
It is deployed on Streamlit: https://oluwadunsi-movie-recommender-main-msgadh.streamlit.app/
